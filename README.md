# Exploratory Data Analysis (EDA) – Customer Shopping Trends

## Project Overview

Performed Exploratory Data Analysis (EDA) on the Customer Shopping Trends dataset using Python. The project focuses on data validation, data cleaning, visualization, and identifying customer purchasing patterns to generate meaningful business insights.

## Objectives

* Validate and clean the dataset.
* Explore customer purchasing behavior.
* Identify trends and relationships.
* Generate business insights through visualizations.

## Dataset

* **Source:** Kaggle – Customer Shopping Trends Dataset
* **Records:** 3,900
* **Features:** 18

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

1. Data Loading
2. Data Validation
3. Data Cleaning
4. Exploratory Data Analysis (EDA)
5. Data Visualization
6. Insight Generation

## Key Analysis

* Univariate Analysis
* Bivariate Analysis
* Multivariate Analysis
* Correlation Analysis
* Customer Segmentation

## Key Insights

* Analyzed customer purchasing behavior.
* Compared spending across different customer segments.
* Identified trends based on categories, seasons, and subscription status.
* Generated business insights for decision-making.
